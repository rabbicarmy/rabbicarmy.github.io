One-time setup: Create a GitHub Personal Access Token

  1. Go to GitHub Token Settings
  2. Make sure public_repo is checked (should be pre-selected)
  3. Click "Generate token" at the bottom
  4. Copy the token and paste it below

Your token is stored only in your browser and never sent anywhere except GitHub.
